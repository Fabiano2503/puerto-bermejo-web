# Cevichería Puerto Bermejo - Fase 1

Página web rápida y optimizada para móviles de una cevichería real. Desarrollada con Astro y Tailwind CSS.

## 🚀 Stack Tecnológico

- **Astro** - Framework
- **Tailwind CSS** - Estilos utilitarios mobile-first
- **JSON local** - Datos del menú en `carta.json`

## ✅ Características Implementadas

- ✅ Diseño mobile-first con Tailwind CSS
- ✅ Menú cargado dinámicamente desde `carta.json`
- ✅ Soporte para precios simples y variantes (personal/mediano/familiar)
- ✅ Botón de WhatsApp siempre visible (header + hero + footer)
- ✅ Mensaje precargado en enlaces de WhatsApp
- ✅ Tipografía clara y grande
- ✅ Secciones bien separadas sin saturación visual
- ✅ Preparado para Fase 2 (carpeta `src/app/`)

## 🎯 Objetivos de UX Cumplidos

- [x] Ver el menú digital claramente
- [x] Acceso rápido a WhatsApp para pedidos
- [x] Información del negocio (ubicación, horario)
- [x] Mejorar experiencia del cliente en el local
- [x] Menú escaneable en menos de 10 segundos
- [x] Enfoque en conversión (WhatsApp)
